# QA CHECKLIST ✅

## User booking flow
- [ ] `/start` открывает русское меню с эмодзи.
- [ ] `✂️ Записаться` запускает пошаговый сценарий.

- [ ] После подтверждения запись видна в `📅 Мои записи`.

## My bookings
- [ ] Карточка записи открывается без ошибок.

- [ ] Кнопка `🆘 Поддержка` показывает статичный контакт.

- [ ] Отмена записи доступна и подтверждается.

## Admin dashboard + actions

- [ ] Карточка записи открывается.
- [ ] Действия (связаться/отменить/комментарий) работают или дают понятный отказ.

- [ ] `👥 Клиенты` открывается.
- [ ] Поиск выдает результаты.
- [ ] Карточка клиента и история открываются.

## Loyalty (if enabled)
- [ ] Экран лояльности открывается.
- [ ] Начисление/списание/скидка не падают.

## Broadcasts (if enabled)
- [ ] `📣 Рассылка` проходит сценарий draft → confirm.

- [ ] `⬅️ Назад` возвращает на предыдущий экран.
